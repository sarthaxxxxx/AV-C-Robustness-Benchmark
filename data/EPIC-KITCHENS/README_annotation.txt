# EPIC-KITCHENS-100 - Auto-Annotations Release

> Release Date: June 2020
>
> Jian Ma, Will Price, Dima Damen (University of Bristol)

This repository contains automatic annotations for EPIC-KITCHENS-100 dataset
[EPIC-KITCHENS](https://epic-kitchens.github.io).

## Videos
All videos are available on a separate [repository](https://doi.org/10.5523/bris.2g1n6qdydwa9u22shpxqzp0t8m). 

## Automatic Annotations
Two sets of automatic annotations are extracted and available for downloads. The first is masks utilising Mask R-CNN model pre-trained on COCO. The second is hand-object correpsondences of active (in-use) objects using the model of [Shan et al (2020)](https://fouheylab.eecs.umich.edu/~dandans/projects/100DOH/).

## Usage
We provide stub code to plot and understand these automatic narrations at our [Github repositories](https://github.com/epic-kitchens) for usage libraries
